# Credit Card Transaction Analysis (SQL Project)

## Overview
This project analyzes credit card transaction data to understand customer spending behavior, income levels, credit scores, and demographic patterns using SQL.

The goal is to identify high-value customers and understand financial behavior trends.

---

## Tools Used
- SQL (SQLite / DB Browser for SQLite)
- GitHub

---

## Key Analyses
- Total transactions and customer base
- Customer spending segmentation
- Age vs spending behavior
- Credit score vs transaction behavior
- Income vs spending analysis

---

## Key Insight
A small group of high-value customers contributes a large portion of total spending, and income and credit score influence spending behavior.

---

## Skills Used
- SQL (JOINs, GROUP BY, CASE WHEN, Aggregations)
- Data cleaning
- Customer segmentation
- Business analysis
